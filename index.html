Welcome to bean-cp project

Bean-cp is object to object mapping library for Java. Library is targeted for Java version 8 to bring the power of lambas to object mapping. Initial API design is already finished, so you may check if it is suitable for you.

Project is still work in progress with no release so far. Expect first release till end of May 2014.
