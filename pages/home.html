Welcome to bean-cp project

Bean-cp is object to object mapping library for Java. Library is targeted for Java version 8 to bring the power of lambas to object mapping.

Version 1.0 is available for download. It will be also available soon in Maven's Central repository. Bean-cp library requires two dependencies: Apache Commons Lang and Javassist.

Get started

API specification is available to read on-line and for download. Get started guide and and user manual coming soon.
